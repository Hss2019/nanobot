## Round 2: Bug 修复 + 完善

### Bug: 配置了 Key 也不能对话
- **根因**: desktop 模式没有 ChannelManager，outbound 消息堆在 MessageBus 里无人消费
- **修复**: `webui/server.py` 在 lifespan 中启动 `_dispatch_outbound()` 异步任务，消费 bus.outbound 并路由到 WebChannel

### Bug: 无 Key 无法启动
- **根因**: `_make_provider()` 找不到 API Key 时调用 `typer.Exit(1)` 直接退出
- **修复**: 新增 `_make_provider_safe()` 捕获 SystemExit，返回占位 LiteLLMProvider；前端显示黄色 banner 引导配置

### Bug: 进程杀不死
- **根因**: 多线程（uvicorn + pywebview + pystray）缺乏统一关闭机制
- **修复**: `desktop/app.py` 重写，添加 `_shutdown()` 集中关闭 + signal handler + atexit + 2s 强制 os._exit

### Enhancement: onboard --yes
- 新增 `-y/--yes` 参数跳过确认提示（refresh 模式）
